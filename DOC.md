# Romario Permana Putra (L200230260)

## 1. Deskripsi Aplikasi
Aplikasi Single Page Application sederhana yang dibuat dengan React (Vite) untuk menampilkan Portofolio. Aplikasi menampilkan tiga bagian utama yaitu:
- Profil (foto, nama, NIM, tentang)
- Keahlian (daftar skill dengan persentase)
- Pengalaman (riwayat organisasi atau pekerjaan)


## 2. Cara Menjalankan (Pengembangan Lokal)
1. buka folder proyek:
cd C:\Users\portofolio
2. Install dependencies:
npm install
3. Jalankan mode development:
npm run dev

## 3. Deployment
Syarat: GitHub repo terhubung dan sudah berisi commit terakhir
1. Import repo ke Vercel
2. Verifikasi hasil deploy
